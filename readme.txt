Task 1 Description — Retention Analytics ML Models

Research and analyze propensity-based Machine Learning models that can be used for retention analytics and customer churn prediction use cases. Study various ML algorithms such as Logistic Regression, Decision Tree, Random Forest, XGBoost, and Neural Networks using publicly available datasets and Kaggle references. Prepare detailed documentation covering algorithm overview, use cases, advantages, disadvantages, comparison, workflow, and suitability for Mercer Superannuation retention analytics scenarios.

Task 2 Description — Databricks MLOps

Research Databricks MLOps concepts including model versioning, model performance monitoring, model bias detection, MLflow, model registry, and Unity Catalog governance. Prepare detailed documentation explaining the MLOps lifecycle, Databricks workflow, monitoring strategies, governance concepts, and enterprise implementation approaches for Machine Learning models used in retention analytics solutions.
